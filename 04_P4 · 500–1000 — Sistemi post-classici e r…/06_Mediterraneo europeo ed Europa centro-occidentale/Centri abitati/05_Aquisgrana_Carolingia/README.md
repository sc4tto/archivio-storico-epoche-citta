# Aquisgrana carolingia

Modulo del Blocco 42, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Residenza favorita di Carlo Magno, con palazzo, aula regia, cappella palatina, terme, officine e rete di tenute fra Reno, Mosa e Ardenne..
