# Auckland portuale, tranviaria e bellica

Modulo del Blocco 91, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Metropoli portuale di Tāmaki Makaurau, sviluppata fra Waitematā e Manukau con tram elettrici, industrie, immigrazione, influenza del 1918 e presenza militare nel Pacifico..
