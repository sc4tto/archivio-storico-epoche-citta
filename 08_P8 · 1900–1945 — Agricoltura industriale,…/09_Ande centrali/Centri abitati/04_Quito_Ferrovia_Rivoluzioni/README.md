# Quito ferroviaria, moderna e rivoluzionaria

Modulo del Blocco 89, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale andina collegata alla costa dalla ferrovia del 1908, dotata di tram, nuovi quartieri e istituzioni, attraversata da conflitti liberali e dalla rivoluzione del 1944..
