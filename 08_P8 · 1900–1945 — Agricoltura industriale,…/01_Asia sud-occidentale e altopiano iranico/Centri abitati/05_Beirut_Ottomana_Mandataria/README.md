# Beirut ottomana, mandataria e indipendente

Modulo del Blocco 81, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Porto levantino passato dalla provincia ottomana alla capitale del Grande Libano sotto mandato francese e poi dello Stato indipendente, segnato da carestia, migrazioni, ferrovie, università, banche e quartieri moderni..
