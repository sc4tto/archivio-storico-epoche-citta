# Tel Aviv–Jaffa nel periodo mandatario

Modulo del Blocco 81, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Nuovo centro urbano ebraico fondato nel 1909 accanto alla città portuale araba di Jaffa, cresciuto durante il Mandato britannico attraverso immigrazione, pianificazione a città-giardino, architettura moderna, industrie e conflitto intercomunitario..
