# Baghdad ottomana, mandataria e monarchica

Modulo del Blocco 81, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale provinciale ottomana conquistata dai britannici nel 1917 e trasformata nella capitale dell’Iraq mandataria e monarchico, con nuove strade, ferrovie, ponti, ministeri e reti petrolifere..
