# Addis Abeba imperiale e occupata

Modulo del Blocco 82, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale imperiale etiope collegata dalla ferrovia a Gibuti, sede di palazzi, mercati, scuole e ministeri, occupata dall’Italia fascista fra 1936 e 1941 e poi restaurata sotto Hailé Selassié..
