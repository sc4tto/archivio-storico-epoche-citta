# Saigon coloniale, industriale e occupata

Modulo del Blocco 85, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale economica della Cocincina francese, porto del delta del Mekong con docks, ferrovie, industrie e quartieri vietnamiti, cinesi ed europei, poi sottoposta al dominio giapponese..
