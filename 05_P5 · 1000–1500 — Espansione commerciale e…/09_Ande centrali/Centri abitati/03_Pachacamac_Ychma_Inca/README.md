# Pachacamac fra Ychma e Inca

Modulo del Blocco 56, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande santuario della valle del Lurín, centro di pellegrinaggio e oracolo, con piramidi con rampa, templi, piazze, depositi, percorsi cerimoniali e trasformazioni inca nel tardo XV secolo..
