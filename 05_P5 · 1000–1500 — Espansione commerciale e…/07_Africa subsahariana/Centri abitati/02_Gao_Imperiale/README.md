# Gao imperiale e Tomb of Askia

Modulo del Blocco 54, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale sul Niger fra regni di Gao e Impero Songhai, con insediamenti gemelli, mercati, moschee in terra, necropoli, porti fluviali e il complesso funerario di Askia Muhammad del 1495..
