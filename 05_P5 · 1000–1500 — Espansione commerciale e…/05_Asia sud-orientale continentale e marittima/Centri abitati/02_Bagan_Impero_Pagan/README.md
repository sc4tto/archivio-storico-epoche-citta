# Bagan, capitale dell’Impero Pagan

Modulo del Blocco 52, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Paesaggio sacro e capitale regionale sul fiume Ayeyarwady, caratterizzato fra XI e XIII secolo da migliaia di templi, stupa, monasteri, villaggi e terre donate..
