# Vijayanagara–Hampi

Modulo del Blocco 50, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale imperiale fondata nel XIV secolo sul Tungabhadra, con centri sacri e reali, bazar, fortificazioni, sistemi idraulici e una vasta regione agricola integrata..
