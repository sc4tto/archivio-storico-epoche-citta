# Lahore ghaznavide e sultaniale

Modulo del Blocco 50, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.

Descrizione: Grande nodo del Punjab fra mondo iranico, valle dell’Indo e pianure gangetiche, sede politica e militare con cittadella, mura, mercati e istituzioni religiose..
