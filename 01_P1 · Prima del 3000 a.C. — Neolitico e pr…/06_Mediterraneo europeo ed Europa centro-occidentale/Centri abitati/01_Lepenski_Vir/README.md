# Lepenski Vir

Modulo del Blocco 07, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Sequenza mesolitica e neolitica delle Porte di Ferro, con attenzione agli edifici trapezoidali e alla fase di trasformazione del VII–VI millennio a.C..
