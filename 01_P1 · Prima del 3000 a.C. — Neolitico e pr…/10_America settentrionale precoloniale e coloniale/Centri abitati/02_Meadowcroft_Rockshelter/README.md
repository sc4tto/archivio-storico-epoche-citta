# Meadowcroft Rockshelter

Modulo del Blocco 11, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Riparo stratificato degli Appalachi occidentali con una lunga sequenza di frequentazioni, focolari, litica, resti botanici e faunistici e connessioni regionali lungo il bacino dell’Ohio..
