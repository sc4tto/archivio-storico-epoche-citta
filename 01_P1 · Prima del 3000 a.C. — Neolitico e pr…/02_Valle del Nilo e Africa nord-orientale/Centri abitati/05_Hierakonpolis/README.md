# Hierakonpolis

Modulo del Blocco 03, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Alto Egitto — grande centro predinastico con abitati, aree produttive e necropoli.
