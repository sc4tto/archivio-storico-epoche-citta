# Zohapilco

Modulo del Blocco 09, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Sito all’aperto frequentato fra circa 5500 e 2200 a.C. sul margine del lago Chalco, importante per lo sfruttamento lacustre, la crescente stabilità insediativa e le fasi iniziali della ceramica nel Bacino del Messico..
