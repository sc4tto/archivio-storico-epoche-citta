# Guilá Naquitz

Modulo del Blocco 09, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Riparo stagionale dell’Arcaico antico, noto per resti botanici eccezionalmente conservati e per le prime fasi della gestione e domesticazione di zucca, zucca-bottiglia e mais..
