# Paloma

Modulo del Blocco 10, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Villaggio preceramico costiero, frequentato per millenni fra lomas, valle e Pacifico, con capanne di canne, pesca intensiva, raccolta, sepolture e crescente stabilità insediativa..
