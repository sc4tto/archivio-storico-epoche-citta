# Huaca Prieta

Modulo del Blocco 10, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Sequenza costiera di lunga durata, dalle frequentazioni pleistoceniche e oloceniche iniziali fino al villaggio preceramico con pesca, raccolta marina, fibre vegetali, tessili e architetture seminterrate..
