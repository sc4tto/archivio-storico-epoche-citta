# Caral-Supe

Modulo del Blocco 10, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro monumentale del Tardo Arcaico delle Ande centrali, con grandi piattaforme, corti circolari incassate, residenze, piazze e reti economiche fra costa, valle e territori interni..
