# Áspero

Modulo del Blocco 10, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro costiero del sistema Caral-Supe con pesca intensiva, edifici monumentali, piattaforme, piazze, residenze e integrazione fra risorse marine, cotone e prodotti delle valli..
