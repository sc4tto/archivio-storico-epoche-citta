# Nanchoc

Modulo del Blocco 10, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso di siti all’aperto con abitazioni leggere, orticoltura precoce, macinazione, colture come zucca, arachide e cotone e, nelle fasi tarde, piccoli canali di irrigazione..
