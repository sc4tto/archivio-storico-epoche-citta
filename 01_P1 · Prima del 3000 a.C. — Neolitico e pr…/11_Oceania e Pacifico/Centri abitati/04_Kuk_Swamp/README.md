# Kuk Swamp

Modulo del Blocco 12, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Paesaggio umido d’altopiano con evidenze di manipolazione del suolo, drenaggio, coltivazione, fossati e trasformazione progressiva verso sistemi agricoli indipendenti..
