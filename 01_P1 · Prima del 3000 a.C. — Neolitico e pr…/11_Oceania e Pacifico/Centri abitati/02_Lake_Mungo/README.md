# Lake Mungo

Modulo del Blocco 12, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.

Descrizione: Paesaggio lacustre pleistocenico con focolari, strumenti, resti alimentari, sepolture e una lunga relazione fra comunità umane, dune lunette e oscillazioni climatiche..
