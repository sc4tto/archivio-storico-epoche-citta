# Ounjougou

Modulo del Blocco 08, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso di oltre cento località con una lunga sequenza paleoambientale e archeologica.
