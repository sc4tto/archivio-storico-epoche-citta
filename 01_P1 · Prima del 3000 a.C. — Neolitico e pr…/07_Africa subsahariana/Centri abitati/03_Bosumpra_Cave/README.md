# Bosumpra Cave

Modulo del Blocco 08, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Riparo con una sequenza di circa 12.500 anni, importante per l’uso di piante, le industrie microlitiche e la comparsa precoce di ceramica nella fascia foresta-savana dell’Africa occidentale..
