# Óc Eo

Modulo del Blocco 30, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande sistema insediativo e portuale del delta del Mekong, attivo soprattutto nei primi secoli d.C., con canali, aree abitative, officine, strutture rituali e contatti a lunga distanza collegati alle formazioni politiche note dalle fonti come Funan..
