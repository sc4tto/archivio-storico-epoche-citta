# Batujaya

Modulo del Blocco 30, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Ampio paesaggio di tumuli, strutture in mattoni, insediamenti e aree rituali nella pianura costiera di Giava occidentale, con fasi dal periodo Buni all’età di Tarumanagara e collegamenti alle rotte del Mar di Giava..
