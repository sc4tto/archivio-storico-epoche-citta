# Angkor Borei

Modulo del Blocco 30, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento fortificato e nodo fluviale occupato dal IV secolo a.C. almeno fino al VII secolo d.C., con mura, fossati, canali, aree abitative, artigianato, iscrizioni e collegamenti verso Óc Eo e Phnom Da..
