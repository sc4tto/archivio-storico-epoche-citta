# Antiochia

Modulo del Blocco 25, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale seleucide fondata alla fine del IV secolo a.C. e grande metropoli romana sull’Oronte, con mura, quartieri, colonnati, acquedotti, terme, ville, mosaici, porto a Seleucia Pieria e il vicino sobborgo di Dafne..
