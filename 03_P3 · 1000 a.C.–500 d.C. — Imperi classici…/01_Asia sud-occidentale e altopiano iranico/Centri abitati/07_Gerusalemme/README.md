# Gerusalemme

Modulo del Blocco 26, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Città d’altura con continuità fra età del Ferro, fasi persiana ed ellenistica, regni asmoneo ed erodiano, dominio romano e tarda antichità, centro politico e religioso di eccezionale densità storica..
