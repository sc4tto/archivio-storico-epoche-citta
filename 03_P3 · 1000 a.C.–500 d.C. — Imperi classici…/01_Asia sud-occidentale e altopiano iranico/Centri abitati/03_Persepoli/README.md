# Persepoli

Modulo del Blocco 25, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso reale achemenide fondato da Dario I intorno al 518 a.C. su una grande terrazza, con Apadana, Porta di tutte le Nazioni, Palazzo di Dario, Sala delle Cento Colonne, rilievi, archivi e infrastrutture di servizio..
