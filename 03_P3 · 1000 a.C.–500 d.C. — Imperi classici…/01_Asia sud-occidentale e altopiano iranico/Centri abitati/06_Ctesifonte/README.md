# Ctesifonte

Modulo del Blocco 26, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale invernale arsacide e poi centro imperiale sasanide sul Tigri, sviluppata di fronte a Seleucia come parte di una vasta metropoli policentrica, commerciale e religiosa..
