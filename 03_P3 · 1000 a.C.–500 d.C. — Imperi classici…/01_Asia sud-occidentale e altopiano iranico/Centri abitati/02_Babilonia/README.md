# Babilonia

Modulo del Blocco 25, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Città mesopotamica di lunga durata, capitale neo-babilonese fra 626 e 539 a.C., con doppia cinta, porte monumentali, via processionale, palazzi, templi, ziggurat, canali e importanti tradizioni matematiche e astronomiche..
