# Petra

Modulo del Blocco 26, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale nabatea in un paesaggio roccioso, collegata alle rotte dell’incenso e dotata di un sistema idraulico sofisticato, tombe rupestri, templi, quartieri e successivi monumenti romani e tardoantichi..
