# Hopewell Mound Group

Modulo del Blocco 35, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Vasto complesso Hopewell con recinto geometrico, grandi tumuli, aree funerarie e straordinarie deposizioni di rame, mica, ossidiana e conchiglie; il sito diede il nome moderno alla tradizione archeologica Hopewell..
