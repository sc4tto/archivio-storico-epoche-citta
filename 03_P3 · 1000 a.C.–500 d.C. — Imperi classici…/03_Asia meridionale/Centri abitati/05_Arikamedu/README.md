# Arikamedu

Modulo del Blocco 28, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento portuale e artigiano sull’estuario dell’Ariyankuppam, attivo soprattutto fra il II secolo a.C. e i primi secoli d.C., con ceramiche locali e mediterranee, anfore, perle di vetro e pietra, tessili e reti marittime dell’Oceano Indiano..
