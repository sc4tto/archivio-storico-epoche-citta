# Pataliputra

Modulo del Blocco 28, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale di Magadha e dei grandi imperi Nanda, Maurya, Shunga e Gupta, sviluppata presso le vie fluviali del Gange e del Son, con fortificazioni lignee, edifici pubblici, quartieri, monasteri e infrastrutture amministrative..
