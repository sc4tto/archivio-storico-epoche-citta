# Huacas de Moche

Modulo del Blocco 34, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso urbano e cerimoniale moche con due grandi huacas in adobe, quartieri residenziali e produttivi, piazze, canali e una ricca documentazione di pittura murale, ceramica, metallurgia e rituali..
