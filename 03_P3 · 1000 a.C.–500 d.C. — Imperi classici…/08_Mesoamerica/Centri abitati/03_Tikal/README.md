# Tikal

Modulo del Blocco 33, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande centro maya delle basse terre, occupato dal Preclassico e divenuto nel P3 una capitale dinastica con templi, palazzi, piazze, sacbé, serbatoi, iscrizioni e relazioni estese fino al Messico centrale..
