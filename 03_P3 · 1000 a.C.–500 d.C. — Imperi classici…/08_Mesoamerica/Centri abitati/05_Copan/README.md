# Copán

Modulo del Blocco 33, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro politico, civile e religioso della valle del Copán, con occupazioni antiche e una dinastia fondata nel 427 d.C.; il modulo P3 segue le origini della capitale classica e i primi programmi monumentali del V secolo..
