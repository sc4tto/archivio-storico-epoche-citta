# Jenne-jeno

Modulo del Blocco 32, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento proto-urbano del delta interno del Niger, occupato dal III secolo a.C., con quartieri in terra, metallurgia, ceramica, agricoltura, pesca e reti regionali distribuite fra più tell e isole alluvionali..
