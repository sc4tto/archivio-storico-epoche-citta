# Daima

Modulo del Blocco 32, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento a tell del bacino del lago Ciad con una lunga sequenza stratigrafica, comprendente nel P3 comunità agricole e pastorali, pesca, ceramica, figurine, sepolture e l’introduzione graduale del ferro..
