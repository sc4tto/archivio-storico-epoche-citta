# Nok–Taruga

Modulo del Blocco 32, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insieme di villaggi e siti produttivi della cultura Nok, noto per terracotte figurate, ceramica, agricoltura e prime evidenze della metallurgia del ferro nella Nigeria centrale..
