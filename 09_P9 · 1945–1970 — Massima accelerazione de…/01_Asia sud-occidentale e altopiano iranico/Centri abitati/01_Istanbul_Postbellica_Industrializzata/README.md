# Istanbul postbellica, industrializzata e migrante

Modulo del Blocco 92, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Metropoli sul Bosforo trasformata da migrazione interna, industria, nuove strade e quartieri informali, segnata dal pogrom del 1955 e dai colpi di Stato..
