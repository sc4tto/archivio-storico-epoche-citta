# Monterrey, industria pesante e metropoli dei lavoratori

Modulo del Blocco 99, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Monterrey fra 1945 e 1970: espansione dell’acciaio, vetro, birra, cemento e manifattura, crescita demografica, conurbazione con San Nicolás, Guadalupe e Santa Catarina, nuovi quartieri operai, università, autostrade, scarsità idrica e persistente abitazione precaria nei tejabanes..
