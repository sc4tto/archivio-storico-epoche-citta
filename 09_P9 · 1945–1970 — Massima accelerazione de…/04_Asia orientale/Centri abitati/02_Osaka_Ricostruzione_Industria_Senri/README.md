# Osaka, ricostruzione industriale e nuova urbanizzazione

Modulo del Blocco 95, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Osaka fra 1945 e 1970: ricostruzione dopo i bombardamenti, riconversione industriale, porto e reti del Keihanshin, espansione suburbana, Senri New Town, inquinamento, subsidenza e Expo 1970..
