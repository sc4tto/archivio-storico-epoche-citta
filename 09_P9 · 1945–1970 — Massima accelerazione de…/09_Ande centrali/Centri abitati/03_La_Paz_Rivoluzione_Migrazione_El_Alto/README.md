# La Paz, rivoluzione nazionale e conurbazione altiplanica

Modulo del Blocco 100, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: La Paz e l’altopiano urbano fra 1945 e 1970: Rivoluzione nazionale del 1952, nazionalizzazione delle miniere, riforma agraria, migrazione aymara e mineraria, crescita di El Alto, mercati, trasporti e servizi in una topografia fortemente diseguale..
