# Johannesburg, apartheid, industria e resistenza

Modulo del Blocco 98, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Johannesburg fra 1945 e 1970: economia mineraria e industriale, istituzionalizzazione dell’apartheid dal 1948, Group Areas Act del 1950, township nere e controllo del lavoro, rimozione forzata di Sophiatown dal 1955, sviluppo di Soweto, Freedom Charter a Kliptown, repressione e organizzazione politica clandestina..
