# Karachi, porto, industria, acqua, mobilità e clima

Modulo del Blocco 105, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Karachi dal 1970 al 25 giugno 2026: porto e industria, migrazioni interne e afghane, crescita delle katchi abadis, violenza politica ed etnica, crisi idrica e fognaria, trasporto informale e BRT, ondate di calore, alluvioni e vulnerabilità costiera..
