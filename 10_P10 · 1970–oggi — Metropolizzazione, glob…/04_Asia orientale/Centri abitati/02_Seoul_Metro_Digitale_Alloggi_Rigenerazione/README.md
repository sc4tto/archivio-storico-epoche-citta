# Seoul, metropolitana, città digitale, alloggi e rigenerazione

Modulo del Blocco 106, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Seoul dal 1970 al 25 giugno 2026: industrializzazione autoritaria, crescita di Gangnam, apertura della metropolitana, democratizzazione, ricostruzione e riqualificazione, riforma del trasporto pubblico, governo digitale, invecchiamento, prezzi immobiliari, alluvioni e politiche climatiche..
