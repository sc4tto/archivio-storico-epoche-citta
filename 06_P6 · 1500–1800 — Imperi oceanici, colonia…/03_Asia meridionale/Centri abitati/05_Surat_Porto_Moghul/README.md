# Surat, porto moghul

Modulo del Blocco 61, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Principale porto moghul dell’India occidentale e punto di imbarco per il pellegrinaggio alla Mecca, con castello, dogane, sarai, compagnie europee e cantieri navali..
