# Manila–Intramuros spagnola

Modulo del Blocco 63, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale fortificata delle Filippine spagnole dal 1571, centro di governo, religione e istruzione, collegato al porto di Cavite, al Parian e al traffico dei galeoni di Acapulco..
