# Città del Messico vicereale

Modulo del Blocco 66, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale della Nuova Spagna costruita sul sito di México-Tenochtitlan, centro amministrativo, ecclesiastico, giudiziario e commerciale, con piazza maggiore, canali, calzadas, mercati e quartieri indigeni..
