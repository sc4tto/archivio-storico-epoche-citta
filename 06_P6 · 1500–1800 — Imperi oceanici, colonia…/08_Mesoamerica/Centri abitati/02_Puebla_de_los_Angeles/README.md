# Puebla de los Ángeles vicereale

Modulo del Blocco 66, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Città fondata nel 1531 lungo la strada fra Veracruz e Città del Messico, celebre per la maglia regolare, le manifatture tessili e ceramiche, i conventi e l’agricoltura cerealicola della valle..
