# Zanzibar Stone Town omanita

Modulo del Blocco 65, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Città commerciale swahili sulla costa occidentale di Unguja, trasformata dopo la conquista omanita del 1698, con forte, moschee, case in pietra corallina, bazar e reti fra Africa, Arabia e India..
