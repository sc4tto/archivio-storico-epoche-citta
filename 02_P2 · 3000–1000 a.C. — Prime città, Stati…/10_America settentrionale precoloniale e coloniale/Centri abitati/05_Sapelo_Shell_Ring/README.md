# Sapelo Island Shell Ring

Modulo del Blocco 23, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso costiero del Tardo Arcaico con anelli di conchiglie, spazi centrali, aree domestiche e attività specializzate, legato alla raccolta di ostriche, pesca, ceramica a tempera di fibre e mobilità tra isole barriera..
