# Abydos

Modulo del Blocco 15, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro sacro dell’Alto Egitto con cimiteri reali protodinastici, recinti monumentali, insediamento, tempio di Osiride e, nel Medio e Nuovo Regno, cappelle, processioni e complessi funerari..
