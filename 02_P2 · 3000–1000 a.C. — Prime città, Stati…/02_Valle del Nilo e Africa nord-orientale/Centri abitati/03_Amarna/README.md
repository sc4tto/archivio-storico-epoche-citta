# Amarna

Modulo del Blocco 15, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale di breve durata fondata da Akhenaten nel XIV secolo a.C., con città centrale, palazzi, templi dell’Aten, quartieri, laboratori, villaggi di lavoratori, necropoli e stele di confine..
