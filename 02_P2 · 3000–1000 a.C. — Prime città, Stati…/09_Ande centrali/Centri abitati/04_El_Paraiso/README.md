# El Paraíso

Modulo del Blocco 22, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande complesso monumentale preceramico della costa centrale, con numerosi edifici, cortili, riempimenti in shicra, economia integrata fra risorse marine, piante coltivate e prodotti della valle..
