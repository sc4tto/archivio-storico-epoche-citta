# Chavín de Huántar

Modulo del Blocco 22, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro cerimoniale d’altopiano con terrazze, piazze, edifici in pietra, gallerie interne, canali e sculture; il modulo P2 si concentra sulle fasi iniziali fra circa 1500 e 1000 a.C..
