# Sechín Bajo

Modulo del Blocco 22, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Lunga sequenza dal Tardo Arcaico al Formativo iniziale, con piazze circolari, piattaforme, edifici in pietra e adobe e ripetute ricostruzioni nel paesaggio monumentale della valle di Casma-Sechín..
