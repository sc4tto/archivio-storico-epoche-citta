# Talepakemalai

Modulo del Blocco 24, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande insediamento Lapita antico su margine lagunare, con ceramica dentellata, pali di abitazioni, pesca, orticoltura, animali introdotti, conchiglia lavorata e ossidiana trasportata attraverso il Mare di Bismarck..
