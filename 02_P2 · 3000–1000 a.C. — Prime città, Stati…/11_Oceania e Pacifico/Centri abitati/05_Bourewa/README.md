# Bourewa

Modulo del Blocco 24, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento costiero Lapita fra i più antichi delle Figi, con ceramica decorata e comune, fosse, resti domestici, pesca, orticoltura e materiali che attestano collegamenti a lunghissima distanza..
