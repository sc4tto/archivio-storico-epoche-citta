# Kamgot

Modulo del Blocco 24, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento Lapita del Bismarck orientale con ceramica dentellata, ossidiana, utensili in conchiglia, pesca, colture tropicali e reti che collegavano Nuova Irlanda, Nuova Britannia e le isole più orientali..
