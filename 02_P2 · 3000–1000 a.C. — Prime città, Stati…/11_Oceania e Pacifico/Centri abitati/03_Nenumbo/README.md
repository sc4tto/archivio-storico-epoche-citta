# Nenumbo

Modulo del Blocco 24, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento Lapita delle isole Santa Cruz con ceramica dentellata, pali, conchiglia lavorata, pesca e collegamenti di lunga distanza, fondamentale per ricostruire il passaggio dall’Oceania vicina all’Oceania lontana..
