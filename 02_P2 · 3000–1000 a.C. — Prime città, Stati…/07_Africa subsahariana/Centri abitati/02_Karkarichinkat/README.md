# Karkarichinkat

Modulo del Blocco 20, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso pastorale e funerario del Tilemsi con bovini domestici, ceramica, macine, primi resti di miglio perlato e pratiche di sepoltura in un paesaggio che divenne progressivamente più arido..
