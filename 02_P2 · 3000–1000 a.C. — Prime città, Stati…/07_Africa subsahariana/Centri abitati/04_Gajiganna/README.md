# Gajiganna

Modulo del Blocco 20, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Sequenza di comunità pastorali e poi più sedentarie nel bacino del lago Ciad, con ceramica decorata, figurine, bestiame, coltivazione del miglio, monticoli d’abitato e sistemi di deposito..
