# Dhar Tichitt

Modulo del Blocco 20, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Complesso di villaggi, recinti e centri regionali in pietra a secco, sviluppato fra Sahara e Sahel con pastorizia, coltivazione del miglio, pesca residuale e crescente differenziazione degli spazi domestici..
