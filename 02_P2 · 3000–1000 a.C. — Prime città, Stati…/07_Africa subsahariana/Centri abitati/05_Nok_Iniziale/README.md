# Nok iniziale

Modulo del Blocco 20, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Fase iniziale della tradizione Nok, dalla metà del II millennio a.C., con villaggi agricoli, miglio, ceramica, macine e prime produzioni in terracotta; la metallurgia del ferro appartiene soprattutto allo sviluppo terminale e successivo..
