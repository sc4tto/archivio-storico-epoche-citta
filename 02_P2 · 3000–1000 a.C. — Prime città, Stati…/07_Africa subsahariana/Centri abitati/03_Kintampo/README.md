# Kintampo

Modulo del Blocco 20, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Tradizione archeologica diffusa fra savana e margine forestale, con villaggi all’aperto e ripari, miglio perlato, palma da olio, animali domestici, ceramica, pietra levigata e reti con la costa..
