# El Manatí

Modulo del Blocco 21, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Luogo rituale presso una sorgente e un ambiente umido, con deposizioni di asce di pietra verde, palle di gomma, busti lignei, pigmenti, resti botanici e ossa, conservati in condizioni anaerobiche..
