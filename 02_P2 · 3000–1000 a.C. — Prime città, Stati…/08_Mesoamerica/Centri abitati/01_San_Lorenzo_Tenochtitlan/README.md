# San Lorenzo Tenochtitlán

Modulo del Blocco 21, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande centro del Formativo iniziale costruito su un altopiano modificato, con piattaforme, lagune artificiali, drenaggi, sculture monumentali e una rete di abitati agricoli nel bacino del Coatzacoalcos..
