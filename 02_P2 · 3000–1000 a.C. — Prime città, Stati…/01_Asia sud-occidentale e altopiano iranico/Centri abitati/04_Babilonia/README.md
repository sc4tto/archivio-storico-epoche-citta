# Babilonia

Modulo del Blocco 13, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro urbano dell’Eufrate attestato dal III millennio a.C. e capitale dell’impero antico-babilonese sotto Hammurabi, con quartieri, templi, archivi, mura, canali e un’importante tradizione scribale..
