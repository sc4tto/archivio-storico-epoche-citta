# Ugarit

Modulo del Blocco 14, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Capitale costiera del Bronzo recente, con palazzo reale, templi, quartieri residenziali, archivi multilingui, scrittura alfabetica cuneiforme e collegamenti commerciali con Mediterraneo, Siria interna ed Egitto..
