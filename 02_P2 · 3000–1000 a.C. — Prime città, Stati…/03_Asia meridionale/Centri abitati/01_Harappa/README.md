# Harappa

Modulo del Blocco 16, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande centro della civiltà dell’Indo sviluppato presso un antico corso del Ravi, con più monticoli fortificati, quartieri artigianali, sistemi di drenaggio, cimiteri e una lunga sequenza dal IV al II millennio a.C..
