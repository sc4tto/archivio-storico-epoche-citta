# Mohenjo-daro

Modulo del Blocco 16, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Metropoli della civiltà dell’Indo sulla riva destra dell’Indo, con città bassa pianificata, quartieri residenziali, numerosi pozzi, drenaggi, edifici pubblici e il complesso del Grande Bagno..
