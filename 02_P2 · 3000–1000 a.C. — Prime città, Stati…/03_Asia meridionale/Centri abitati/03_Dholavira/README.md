# Dholavira

Modulo del Blocco 16, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro meridionale della civiltà harappana in ambiente arido, con città fortificata, castello, bailey, città media e bassa, grande area cerimoniale, cimitero e un complesso sistema di raccolta e conservazione dell’acqua..
