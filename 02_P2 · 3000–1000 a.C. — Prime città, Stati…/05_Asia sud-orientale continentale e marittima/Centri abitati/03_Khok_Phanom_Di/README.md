# Khok Phanom Di

Modulo del Blocco 18, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Tumulo costiero stratificato presso l’antico estuario del Bang Pakong, occupato nel II millennio a.C., con pesca, molluschi, riso, produzione ceramica, ornamenti di conchiglia e 154 sepolture..
