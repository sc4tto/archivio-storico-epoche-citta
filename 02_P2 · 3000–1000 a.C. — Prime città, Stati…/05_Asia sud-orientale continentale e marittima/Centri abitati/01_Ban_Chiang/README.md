# Ban Chiang

Modulo del Blocco 18, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande tumulo preistorico della Thailandia nord-orientale con abitazioni, campi umidi, ceramica, sepolture e produzione di bronzo nel tardo II millennio a.C..
