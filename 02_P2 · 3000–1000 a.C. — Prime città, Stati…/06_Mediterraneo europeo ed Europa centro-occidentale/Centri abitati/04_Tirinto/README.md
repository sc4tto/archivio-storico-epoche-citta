# Tirinto

Modulo del Blocco 19, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Cittadella micenea con mura ciclopiche eccezionalmente massicce, gallerie interne, palazzo a megaron, rampe, porte, accessi all’acqua e una città bassa collegata al golfo Argolico..
