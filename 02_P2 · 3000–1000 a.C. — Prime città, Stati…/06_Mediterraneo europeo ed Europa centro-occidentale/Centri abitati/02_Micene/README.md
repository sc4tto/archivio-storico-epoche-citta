# Micene

Modulo del Blocco 19, generato da sette JSON nella cartella `dati`.

- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Grande centro palaziale miceneo con mura ciclopiche, Porta dei Leoni, palazzo, circoli funerari, tombe a tholos, quartieri artigiani e reti regionali e mediterranee..
