# Londra industriale e vittoriana

Modulo del Blocco 75, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Metropoli imperiale sul Tamigi, trasformata nel XIX secolo da docks, ferrovie, crescita demografica, reti fognarie, nuove istituzioni e forti disuguaglianze fra West End, City ed East End..
