# Sydney coloniale, penale e portuale

Modulo del Blocco 80, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Insediamento Gadigal/Eora trasformato dalla colonia penale britannica del 1788 in porto, capitale coloniale del New South Wales e città commerciale con docks, mercati, caserme, ferrovie e quartieri operai..
