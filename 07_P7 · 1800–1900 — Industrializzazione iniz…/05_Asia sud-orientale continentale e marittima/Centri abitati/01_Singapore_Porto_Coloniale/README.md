# Singapore, porto coloniale e città migrante

Modulo del Blocco 74, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Porto franco britannico fondato nel 1819, nodo dello stretto e delle rotte fra Oceano Indiano e Mar Cinese, cresciuto attraverso quartieri mercantili, migrazioni asiatiche, magazzini, cantieri e infrastrutture coloniali..
