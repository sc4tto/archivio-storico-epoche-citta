# Saigon–Gia Định coloniale

Modulo del Blocco 74, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Centro del delta meridionale vietnamita conquistato dalla Francia nel 1859 e trasformato in capitale della Cocincina coloniale, porto di riso, città di canali, boulevard, caserme, edifici amministrativi e quartieri sino-vietnamiti..
