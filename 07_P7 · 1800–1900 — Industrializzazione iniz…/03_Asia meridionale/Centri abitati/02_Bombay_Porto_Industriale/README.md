# Bombay portuale e industriale

Modulo del Blocco 72, generato da sette JSON nella cartella `dati`.

- Tutte le pagine puntano al CSS e al JavaScript centrali della repository.
- `APRI_CENTRO.bat`: apre il modulo.
- `RIGENERA_CENTRO.bat`: rigenera le pagine.
- `VERIFICA_CENTRO.bat`: controlla JSON, collegamenti e grafi.

Descrizione: Porto della costa occidentale trasformato da bonifiche, docks, ferrovia, commercio del cotone e filature, con forti migrazioni e disuguaglianze abitative..
